# ETF 분석 프레임워크 — 핵심 개념

## 1. 매크로 환경 연동 분석

### 1.1 레짐 분류 (2×2 매트릭스)

|               | 성장 가속                          | 성장 둔화                              |
|---------------|------------------------------------|-----------------------------------------|
| **인플레 상승** | 실물자산, 원자재, 에너지, 단기채    | 금, 원자재, TIPS, 방어주 (스태그플레이션) |
| **인플레 하락** | 성장주, 기술주, 장기채 (골디락스)   | 장기국채, 유틸리티, 배당주, 퀄리티 팩터   |

### 1.2 금리 사이클별 ETF 행동 패턴

```
금리 인상 후기 → 정점 → 인하 초기 → 인하 후기
     │             │          │           │
     ▼             ▼          ▼           ▼
  단기채/변동금리  장기국채    성장주      경기순환주
  에너지/금융      금/유틸    기술/혁신    소형주/EM
  가치주          퀄리티팩터   리츠        하이일드
```

### 1.3 레짐 판단 신호

| 축      | 지표                                              |
|---------|--------------------------------------------------|
| 성장    | ISM PMI 방향, 실질 GDP 추세, 고용 데이터 추세       |
| 인플레  | CPI/PCE 추세, 기대인플레(BEI), 원자재 가격 방향     |
| 유동성  | 연준 B/S 변화, 역레포 잔고, 신용 스프레드           |
| 시장내부 | 수익률 곡선 기울기, 하이일드 스프레드, 시장 폭(breadth) |

> 핵심: "지금 어떤 레짐인가"보다 **"다음 레짐으로 언제 전환되는가"**가 더 가치 있는 질문.
> 레짐 전환 시점에서 가장 큰 초과수익/손실이 발생한다.

## 2. 포트폴리오 레벨 분석

### 2.1 상관관계 블록 구조

대부분의 ETF는 몇 개의 "상관 블록"에 묶인다:

**블록 A — 위험선호 (Risk-On)**
- 미국 대형 성장주, 소형주, 신흥국 주식, 하이일드 채권, 리츠
- 위기 시 상관관계가 동시에 1로 수렴

**블록 B — 안전자산**
- 장기국채, 단기국채, 금
- 주식 하락 시 음의 상관 또는 무상관

**블록 C — 대안적 분산**
- 원자재, TIPS, 관리형 선물, 변동금리 대출
- 인플레 레짐에서 A, B 모두와 다르게 행동

### 2.2 진짜 분산 vs 가짜 분산

| 가짜 분산 (평시 낮은 상관, 위기 시 수렴)  | 진짜 분산 (위기 시에도 유지)          |
|------------------------------------------|--------------------------------------|
| 미국주식 + 유럽주식 + EM 주식             | 주식 + 장기국채 (인플레 안정 시)       |
| 대형성장 + 대형가치                       | 주식 + 금                            |
| 주식 + 하이일드                           | 주식 + 관리형 선물                    |
| 섹터 ETF 10개 조합                        | 주식 + 원자재 (인플레 레짐)           |

검증 방법: 평상시 상관관계가 아닌 **꼬리 상관관계**(tail correlation) — 하락 5% 이상 구간에서의 동조 행동을 확인.

### 2.3 레짐-적응형 포트폴리오 설계

```
코어 (60-70%)          레짐에 덜 민감한 구조적 분산
├── 글로벌 주식 (폭넓은 시장)
├── 중기 국채
└── 금

새틀라이트 (20-30%)    현재 레짐에 맞는 틸트
├── 레짐 판단에 따라 교체
└── 예: 인플레 상승기 → 원자재/에너지 ETF 비중 ↑

꼬리 보호 (5-10%)      레짐 전환 시 보험
├── 장기국채 (디플레 쇼크 대비)
├── 금 (다목적 헤지)
└── 관리형 선물 (추세추종)
```

### 2.4 레짐 전환 시 리밸런싱 트리거

| 신호                           | 행동                                       |
|-------------------------------|-------------------------------------------|
| 수익률곡선 역전 해소 + PMI 반등 | 새틀라이트를 경기순환주/소형주로 틸트         |
| BEI 상승 + 원자재 상승 추세     | 원자재/TIPS 비중 확대, 장기채 축소           |
| 신용 스프레드 급확대            | 새틀라이트 축소, 꼬리보호 확대, 현금 비중 ↑  |
| VIX 급등 + 시장 폭 급축소      | 코어 유지, 리밸런싱 매수 기회 탐색           |

## 3. ETF 비교 분석 핵심 축 (기존 앱 보완 관점)

1. **구조적 차이** — 같은 테마라도 지수 구성 방법론, 리밸런싱 주기가 다름
2. **보이지 않는 비용** — TER 외에 스프레드, 추적오차, 환헤지 비용, 세금 구조
3. **노출의 실질 내용** — 보유종목 겹침률, 상위 10 집중도, 섹터/지역 분포
4. **유동성과 규모** — AUM 작으면 괴리율 문제, 거래량 낮으면 진입/이탈 어려움
5. **시간에 따른 행동 특성** — MDD, 샤프/소르티노, 특정 매크로 환경에서의 상대 성과
